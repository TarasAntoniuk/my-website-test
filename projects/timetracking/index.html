🕐 Time Tracking System

Enterprise-grade employee time tracking system built with modern Java technologies. This system provides real-time presence monitoring for security teams and automated timesheet calculation for HR departments.

Key Features

  • Real-Time Presence Monitoring: Track which employees are currently in the building at any given time
  • Check-In/Check-Out Logging: Record all employee entries and exits with precise timestamps
  • Automated Timesheet Calculation: Generate detailed timesheets with daily work hours breakdown
  • RESTful API: Full-featured API with comprehensive OpenAPI documentation
  • Production-Ready: Deployed with SSL, proper error handling, and audit logging

Technology Stack

Java 21 Spring Boot 3.x PostgreSQL JPA / Hibernate REST API OpenAPI 3.0 Docker SSL/HTTPS
